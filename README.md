# react-native-ethers-crypto

Native cryptographic functions for ethers v6

## Installation

```sh
npm install react-native-ethers-crypto
```

## Usage

```js
import { multiply } from 'react-native-ethers-crypto';

// ...

const result = await multiply(3, 7);
```
